# Archivos .tar.gz:
## Comprimir: 
```sh
tar -czvf archivo.tar.gz /dir/a/comprimir/
```
## Descomprimir: 
```
tar -xzvf archivo.tar.gz
```

# Archivos .tar:
## Comprimir: 
```
tar -cvf archivo.tar /dir/a/comprimir/
```
## Descomprimir: 
```
tar -xvf archivo.tar
```

# Archivos .gz:
## Comprimir: 
```
gzip -9 archivo.php
```
## Descomprimir: 
```
gzip -d archivo.php.gz
```

# Archivos .zip:
## Comprimir: 
```
zip archivo.zip /dir/a/comprimir/
```
## Descomprimir: 
```
unzip archivo.zip
```
